Button

To use the button add class .btn & other class like code .btn-* here star is the colors primary & etc.

To use the outline button add class .btn & other class like code .btn-*-outline here star is the colors primary & etc.

To disable the button add class .disabled or use the disabled attribute to the element.

To focus the button add class .focus or use the focus attribute to the element.

To make perfect slice on the button top right o left use class .bg-color-sliced-top/bottom-right/left to the element.

To toggle the button add attibute data-toggle="button" to toggle buittons active state & if pre toggling add .active class & aria-pressed="true" to the element.
